# trabalho-logica.
Um breve parágrafo descrevendo o propósito geral deste repositório, que armazena alguns códigos simples feitos no depurador GDB.

Este repositório está organizado da seguinte forma:
cadastro-alunos/: Contém os arquivos relacionados ao programa de cadastro de alunos.
cadastro-produtos/: Contém os arquivos relacionados ao programa de cadastro de produtos.
tabuada/: Contém os arquivos relacionados ao programa que gera a tabuada.
 Códigos desenvolvidos no GDB
1. Cadastro de Alunos
Descrição: Este código simula um sistema básico de cadastro de alunos, onde é possível adicionar informações como nome, idade, peso e altura, colocando os dados dos alunos pedidos. O programa foi desenvolvido para fins de aprendizado e depuração com o GDB.
2. Cadastro de Produtos
Descrição: Este programa gerencia um cadastro simples de produtos. Ele permite adicionar o nome, o código e o código de barras. Foi utilizado o GDB para identificar e corrigir possíveis erros de lógica e de memória, você irá colocar os dados do produto. O código tem um problema, não está rodando na hora, ele demora um pouco.
3. Tabuada
Descrição: Um programa simples que solicita um número ao usuário e, em seguida, exibe a tabuada desse número. O uso do GDB foi essencial para entender o fluxo de execução dos laços de repetição e o comportamento das variáveis.
